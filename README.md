## ✨ Overview

In vanilla Minecraft an item frame's border is always there, boxing in whatever
you put on display. **InvisibleFrames** lets you hide that border so the item
appears to float on its own — perfect for item shops, decorations, and builds.

**Hold shears and right‑click an item frame** to make it invisible. Knock the item back out later and the
frame reappears on its own, so you never end up with an invisible empty frame you can't find.

## ✂️ How to Use

| Action | Result |
|---|---|
| **Right‑click** an item frame with shears | The frame turns invisible |
| **Knock the item out** of an invisible frame | The frame becomes visible again |

That's it.

## 🚀 Features

| | Feature                                                                       |
|---|-------------------------------------------------------------------------------|
| 🌟 | **Glow frames too** — works on glowing item frames as well                    |
| 👁️ | **Auto‑reveal** — a frame reappears once its item is knocked out, however it happens (hit, projectile, explosion…) |
| ⛏️ | **Durability consumption** — shears wear in Survival, just like real tools     |
| ✨ | **Enchantment‑aware** — respects the *Unbreaking* enchantment                  |
| 📊 | **Vanilla statistics** — updates your *Used* counter for the shears            |

## Installation

1. Download the latest `InvisibleFrames.jar` from the [Releases](https://github.com/Mrflaxe/InvisibleFrames/releases) page.
2. Drop it into your server's `plugins/` folder.
3. Restart the server.

## 🧩 Compatibility

- **Server software:** Spigot, Paper, and forks
- **Minecraft versions:** `1.16` and up, including the latest releases (26.1.2)
- **Dependencies:** none

The plugin ships per‑version modules and automatically selects the right one for
your server at startup, so a single jar works across the full version range.
